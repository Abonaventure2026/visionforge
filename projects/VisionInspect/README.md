
# VisionInspect – 工业视觉全场景检测系统

[![License](https://img.shields.io/badge/License-Commercial%20Proprietary-red.svg)](LICENSE)
[![C++](https://img.shields.io/badge/C%2B%2B-17-blue.svg)](https://isocpp.org/)
[![Qt](https://img.shields.io/badge/Qt-6.2+-green.svg)](https://qt.io/)
[![OpenCV](https://img.shields.io/badge/OpenCV-4.5+-optional-blue.svg)](https://opencv.org/)

**VisionInspect** 是一款面向工业质检的 **机器视觉全场景检测系统**，集成 **相机标定**、**2D/3D检测**、**缺陷识别**、**尺寸测量** 等核心功能，适配半导体、消费电子、汽车零部件等精密制造场景。

---

## 核心特性

- **轻量化设计**：核心算法纯 C++17 实现，零第三方依赖（OpenCV 可选）
- **多场景覆盖**：标定、2D缺陷检测、3D检测、尺寸测量
- **解耦式架构**：标定、检测、应用三大模块独立封装
- **实时性保障**：标定≤2s、检测≤2s、全流程≤3s
- **跨平台支持**：Windows/Linux，Qt6 + OpenMP 加速
- **工业输出**：深度图、PLY点云、CSV报表

## 依赖环境

- **编译器**：GCC 9+ / Clang 10+ / MSVC 2019+ （支持C++17）
- **Qt**：6.2 或更高（Core, Gui, Widgets, OpenGL, OpenGLWidgets）
- **OpenCV**：可选（用于对比测试，默认关闭）
- **OpenMP**：可选，用于多线程加速
- **CMake**：3.20+

> 注：本项目核心算法完全不依赖 OpenCV，纯 C++ 标准库实现。

## 编译与运行

```bash
# 1. 生成工程
cd VisionInspect

# 2. 创建构建目录
mkdir build && cd build

# 3. 配置CMake（默认不启用OpenCV）
cmake .. -DUSE_OPENCV=OFF -DUSE_OPENMP=ON

# 4. 编译
make -j$(nproc)

# 5. 运行
./run.sh   # 或 ./VisionInspect

# 若需启用 OpenCV 对比功能：
cmake .. -DUSE_OPENCV=ON
```

## 使用流程

1. 加载图像：点击「加载图像」，选择测试图像
2. 标定：采集多张标靶图像，点击「标定」完成相机标定
3. 检测：点击「执行检测」，自动完成缺陷识别
4. 导出结果：
   - 保存结果图像
   - 导出 CSV 报表

## 性能指标（参考硬件配置）

- 标定精度：≤ 0.5 像素（重投影误差）
- 检测精度：≤ ±0.05mm（2D），≤ ±0.02mm（3D）
- 处理速度：标定≤2s，检测≤2s

## 扩展与定制

- 标定模块：支持单目、双目、手眼标定
- 检测算法：支持 2D 缺陷、尺寸测量、3D 检测
- 深度学习：预留 ONNX Runtime / NCNN 接口
