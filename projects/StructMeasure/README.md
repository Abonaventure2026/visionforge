
# StructMeasure – 工业结构件智能测量系统

[![License](https://img.shields.io/badge/License-Commercial%20Proprietary-red.svg)](LICENSE)
[![C++](https://img.shields.io/badge/C%2B%2B-17-blue.svg)](https://isocpp.org/)
[![Qt](https://img.shields.io/badge/Qt-6.2+-green.svg)](https://qt.io/)
[![OpenCV](https://img.shields.io/badge/OpenCV-4.5+-optional-blue.svg)](https://opencv.org/)

**StructMeasure** 是一款面向工业X光/视觉检测的 **结构件几何尺寸与锥角测量系统**，集成 **多级图像预处理**、**鲁棒轮廓提取**、**RANSAC分段拟合** 及 **全套工业指标计算**，适配锥体、柱体等结构件的在线/离线测量。

---

## 核心特性

- **高精度边缘提取**：Canny梯度边缘优先 + Otsu自适应阈值 + 5×5高斯镜像平滑
- **鲁棒轮廓分割**：形态学闭运算 + 最大连通域过滤 + 左右轮廓双路RANSAC拟合
- **全面测量指标**：总长、顶宽、底宽、左锥角、右锥角、对称度（左右锥角差）
- **可视化标注开关**：边界线（蓝）、中线（绿）、横端线（红）、数值标注可独立开关
- **标定与导出**：像素/毫米标定系数，CSV报表导出，结果图像保存
- **纯C++实现**：零第三方依赖（OpenCV可选），跨平台（Windows/Linux）

## 依赖环境
编译器：GCC 9+ / Clang 10+ / MSVC 2019+ （支持C++17）

Qt：6.2 或更高（Core, Gui, Widgets）

OpenCV：可选（用于对比测试，不强制）

CMake：3.20+

注：本项目核心算法完全不依赖 OpenCV，纯 C++ 标准库 + Qt 图像操作实现。

## 编译与运行
```bash
# 1. 生成工程（若未使用 structuremeasure_build.py）
# 若已生成，直接进入项目目录
cd StructMeasure

# 2. 创建构建目录
mkdir build && cd build

# 3. 配置CMake（默认不启用OpenCV）
cmake .. 

# 4. 编译
make -j$(nproc)

# 5. 运行
./StructMeasure
```

若需启用 OpenCV 对比功能（需已安装 OpenCV）：
```bash
cmake .. -DUSE_OPENCV=ON
```

## 使用流程

1. **加载图像**：点击「加载图像」，选择结构件X光或视觉图像（支持 BMP/PNG/JPG）
2. **标定设置**：输入像素/毫米标定系数（默认 1.0）
3. **执行处理**：点击「处理图像」，自动完成：
   - 图像预处理 → 轮廓提取 → 左右拟合 → 测量计算
   - 结果图像自动显示，标注可开关控制
4. **结果查看**：左侧面板显示测量报告（总长、顶宽、底宽、左右锥角、对称度等）
5. **导出结果**：
   - 保存结果图：保存带标注的图像
   - 导出 CSV：导出所有测量指标到 CSV 文件

## 标注开关

- **边界线（蓝）**：显示左右拟合轮廓线
- **中线（绿）**：显示几何中心垂线
- **横端线（红）**：显示上下端横线
- **数值标注**：在图像上叠加长度、角度等数值

## 性能指标（参考硬件配置）
测量精度：亚像素级（取决于图像质量）
处理速度：< 200ms（1080p图像，无OpenCV）
适用工件：锥体、柱体等规则结构件

## 扩展与定制
标定模块：可通过 scaleEdit 调整像素/毫米比例，适应不同成像系统
算法参数：可修改 StructureMeasure.cpp 中的 RANSAC 迭代次数、距离阈值等
新增指标：在 Result 结构体中添加字段，并在 process 中计算即可
