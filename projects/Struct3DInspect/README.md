
# Struct3DInspect – LCOS结构光3D工业检测系统

[![License](https://img.shields.io/badge/License-Commercial%20Proprietary-red.svg)](LICENSE)
[![C++](https://img.shields.io/badge/C%2B%2B-17-blue.svg)](https://isocpp.org/)
[![Qt](https://img.shields.io/badge/Qt-6.2+-green.svg)](https://qt.io/)
[![OpenCV](https://img.shields.io/badge/OpenCV-4.5+-blue.svg)](https://opencv.org/)

**Struct3DInspect** 是一款面向高端工业质检的 **3D 结构光在线检测系统**，集成 **LCOS-LED 高精度投影**、**格雷码+多步相移** 三维重建、**三角测量** 点云生成及 **尺寸测量** 功能，适配半导体、消费电子、新能源等精密制造场景。

---

## 核心特性

- **高精度全域成像**：基于 SXGA-3DM LCOS（1280×1024）与 ASTRI LED 驱动，微米级深度精度
- **多算法融合解相位**：格雷码粗定位 + 多频相移精算 + 多频外差抗噪，鲁棒性行业领先
- **实时三维重建**：支持离线测试与在线产线（帧率≥30fps，依赖硬件）
- **完整工业输出**：深度图（PNG/TIFF）、PLY点云、CSV尺寸报表、检测报告（PDF预留）
- **模块化标定**：相机/投影内外参标定，支持动态配置
- **AI ISP 预留接口**：可集成深度学习缺陷检测（需自行训练）
- **跨平台**：Windows/Linux，Qt6 + OpenCV + OpenMP 加速

## 依赖环境

- **编译器**：GCC 9+ / Clang 10+ / MSVC 2019+ （支持C++17）
- **Qt**：6.2 或更高（Core, Gui, Widgets, OpenGL, OpenGLWidgets）
- **OpenCV**：4.5 或更高（core, imgproc, highgui）
- **OpenMP**：可选，用于多线程加速
- **CMake**：3.22+

> 注：本项目**零重型第三方依赖**（除OpenCV外），纯C++标准库实现核心算法。

## 编译与运行

```bash
# 1. 生成工程（若未使用 slight3d_build.py）
# 若已生成，直接进入项目目录
cd Struct3DInspect

# 2. 创建构建目录
mkdir build && cd build

# 3. 配置CMake（可根据需要开关选项）
cmake .. -DUSE_OPENCV=ON -DUSE_OPENMP=ON -DENABLE_AI_ISP=OFF

# 4. 编译
make -j8

# 5. 运行
./Struct3DInspect

## 测试数据生成
项目内置 gen_3d_test.py脚本，可生成模拟工件结构光序列（无需硬件）：
```bash
python gen_3d_test.py--output ./assets/capture
```
默认生成：
格雷码正负帧（8位 → 16张）
四步相移（4张）
全黑、全白基准（2张）
总计 22张 测试图，包含梯形工件、反光光斑、高斯噪声。

## 使用流程
加载序列：点击「加载拍摄序列」，选择 assets/capture 文件夹
设置参数：调整基线距离、像素标定系数、格雷码位数（与生成时一致）
执行重建：点击「3D重建测量」，自动完成：
格雷码解码 → 包裹相位 → 绝对相位 → 点云重建
自动计算工件长、宽、厚
结果导出：
保存深度图（灰度）
保存 PLY 点云（可用 Meshlab 查看）
导出 CSV 尺寸报表
投影条纹生成：点击「生成投影条纹」可输出供LCOS直接投射的PNG图片

## 性能指标（参考硬件配置）
重建精度：≤ 50 μm（依赖标定与投影质量）
单次重建时间：< 2s（含全流程，OpenMP加速）
适用工件尺寸：20mm ~ 500mm（可调）

## 扩展与定制
标定模块：提供相机与投影仪标定接口（src/calib/），支持棋盘格/圆点格。
AI ISP：预留 ENABLE_AI_ISP 宏，可嵌入NCNN/ONNX Runtime实现缺陷分类。
多机同步：RS422总线支持多设备级联，扩展检测幅面。
